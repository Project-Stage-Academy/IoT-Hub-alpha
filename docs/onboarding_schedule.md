# Internship Onboarding Schedule: Week 1

This schedule outlines the intensive training sessions designed to synchronize the team
on the project's technology stack, development standards, and Python/Django best practices.

---

## 1. Weekly Overview (Updated)

| Day | Focus Area | Goal |
| :--- | :--- | :--- |
| Day 1 | SDLC & Orientation | Understand internship goals, team roles, repo layout. Set up accounts, clone repo, install toolchain, basic Linux commands, and SSH keys. |
| Day 2 | Git & Python Environment | Learn Git workflows, Python 3.x environment management. Create virtualenv, pip, basic packaging, type hints. Practical: prepare starter branch and CI skeleton. |
| Day 3 | Modern Python Practices | Deep dive into modern Python features: asyncio overview, context managers, logging best practices. Practical: refactor small module to use async or typed interfaces. |
| Day 4 | Django Fundamentals | Learn Django basics: Models, ORM, Views, URL routing, settings. Practical: scaffold Django project and create Device model. |
| Day 5 | Database & Testing | PostgreSQL basics and Django integration: migrations, indexes, simple queries. Unit testing in Python with `unittest` and `pytest`. Practical: configure Postgres, run migrations, insert sample data, write unit tests. |

---

## 2. Daily Sessions

### Day 1: SDLC & Orientation
* **Topics:** Internship goals, team roles, repo layout, account setup, basic Linux commands, SSH keys.
* **Hands-on:** Clone the repository, set up SSH keys, install toolchain, and verify local environment.
* **Owner:** Project Owner / DevOps Lead

---

### Day 2: Git & Python Environment
* **Topics:** Git workflows, Python 3.x environment setup, virtualenv, pip, basic packaging, type hints.
* **Hands-on:** Prepare starter branch, set up CI skeleton.
* **Owner:** Backend Lead / DevOps Lead

---

### Day 3: Modern Python Practices
* **Topics:** Asyncio overview, context managers, logging best practices.
* **Hands-on:** Refactor a small module to use async functions or typed interfaces.
* **Owner:** Backend Lead

---

### Day 4: Django Fundamentals
* **Topics:** Models, ORM, Views, URL routing, settings.
* **Hands-on:** Scaffold Django project and create a Device model.
* **Owner:** Backend Lead

---

### Day 5: Database & Testing
* **Topics:** PostgreSQL basics, Django integration, migrations, indexes, simple queries. Unit testing with `unittest` and `pytest`.
* **Hands-on:** Configure Postgres, run migrations, insert sample data, write unit tests for models and endpoints.
* **Owner:** Backend Lead / DevOps Lead

---

## 3. Preparation Checklist

Before Day 1, please ensure you have the following installed:

1. **Docker & Docker Compose** (latest version)
2. **Git** (configured with your GitHub account)
3. **Python 3.12+** (for IDE linting and local development)
